gurb

Replaces GRUB with systemd-boot under shim on Ubuntu.
Manages Secure Boot signing, MOK enrollment, and UKI builds—one package, one command. 

# add repo key + source
curl -fsSL https://apt.dab.ac/dab.ac.gpg |
  sudo tee /usr/share/keyrings/dab.ac.gpg > /dev/null
echo 'Types: deb
URIs: https://apt.dab.ac
Suites: ./
Signed-By: /usr/share/keyrings/dab.ac.gpg' |
  sudo tee /etc/apt/sources.list.d/dab.ac.sources > /dev/null

Then: sudo apt update && sudo apt install gurb
